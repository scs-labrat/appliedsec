## What is ALUSKORT?

ALUSKORT is an autonomous SOC (Security Operations Centre) agent that ingests security alerts from SIEM platforms, enriches them with threat intelligence from multiple sources, reasons about them using LLM-powered agents, and orchestrates response actions — all tracked by an immutable, hash-chained audit trail.

The name reflects the project's Icelandic-inspired nomenclature for its cloud-neutral security reasoning architecture.

## Getting Started

```bash
# 1. Install
pip install -e ".[dev]"

# 2. Start infrastructure
docker-compose up -d

# 3. Initialize databases
python infra/scripts/create_kafka_topics.py
python infra/scripts/init_neo4j.py
python infra/scripts/init_qdrant.py

# 4. Run tests
python -m pytest tests/ -v

# 5. Set ANTHROPIC_API_KEY for LLM features
export ANTHROPIC_API_KEY=sk-ant-...
```
