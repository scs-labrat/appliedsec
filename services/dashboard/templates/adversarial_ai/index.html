CTEM Integration Pipeline

Garak: LLM Probes → ctem.raw.garak → CTEM Normaliser → ctem.normalized → ATLAS Mapper

ART: Robustness Tests → ctem.raw.art → CTEM Normaliser → ctem.normalized → ATLAS Mapper

Both tools publish findings to Kafka topics. The CTEM normaliser assigns consequence-weighted severity using the Purdue model zone mappings. The ATLAS Mapper correlates findings to MITRE ATLAS techniques.
